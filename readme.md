## 运行环境

1. 安装mysql
2. 安装node.js
3. Navica2(mysql图形化软件)或者其他
3. 安装jdk环境(不确定是否需要)
4. 安装git(不确定是否需要)
## 参数配置
1. 数据库配置

	`AppRenameServe/db/config.js`

	打开这个文件, 修改数据库配置信息
2. 运行
 
	`AppRename`和`AppRenameServe`

	分别进入这个文件的根目录, shift + 鼠标右键, 选择'在此处打开powershell窗口', 输入命令行:

	`npm install`

	等待安装完成

	`npm start`

3. 打开浏览器, 在浏览器中输入http://localhost:4200/
